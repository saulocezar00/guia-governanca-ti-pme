# 🏛️ Adaptação Prática: ITIL e COBIT para PMEs

Frameworks tradicionais foram desenhados para grandes corporações. Nosso trabalho foi filtrar o essencial para empresas menores.

## ⚖️ O Filtro do COBIT (Foco em Estratégia)
Em vez de adotar os dezenas de objetivos de controle do COBIT, a PME deve focar em apenas dois pilares:
1.  **Alinhamento Estratégico:** O software que está sendo desenvolvido resolve um problema real de faturamento ou redução de custo da empresa?
2.  **Gestão de Riscos:** Onde estão os dados dos clientes? Existe backup? Quem tem acesso à produção?

## 🛠️ O Filtro do ITIL (Foco em Operação)
Para a Engenharia de Software moderna, reduzimos o ITIL a duas práticas essenciais integradas ao ecossistema Git:
* **Gerenciamento de Incidentes:** Centralizado nas *Issues* do GitHub com tags claras (`bug`, `critical`).
* **Gerenciamento de Mudanças:** Todo código novo passa obrigatoriamente por um *Pull Request* com aprovação de outro par (Peer Review), garantindo que nenhuma alteração vá para produção sem validação.
